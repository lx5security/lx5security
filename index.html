About

LX5 Security is a personal cybersecurity archive focused on learning, experimentation, and defensive-first thinking. This site documents concepts, projects, and research as skills evolve.

Focus Areas

  • Cryptography — fundamentals, stream ciphers, and theory
  • Web Security — OWASP concepts, secure design, testing
  • Blue Team — detection, logging, incident thinking
  • Cloud Basics — security concepts for modern environments
