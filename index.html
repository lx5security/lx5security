Building a File Integrity Monitor (FIM) in Rust

File Integrity Monitoring (FIM) is a foundational defensive security control, but it’s often treated as a black box. This post documents how I built LX5 Aegis, a lightweight Linux FIM agent written in Rust, to understand how integrity monitoring works internally.

The project uses cryptographic SHA-256 baselines, real-time filesystem events, and systemd integration to detect unauthorized changes to critical system files.

The focus is on design decisions, security trade-offs, and lessons learned from building a defensive security tool from first principles.
